SLAP Maps

Simple Library for Automating Publication of Map Services

What is SLAP?

  • A command line tool for publishing map services
  • A set of Python modules

What problems does SLAP solve?

  • Scale - tens or hundreds of services
  • Continuous Integration
  • Managing services across different environments
