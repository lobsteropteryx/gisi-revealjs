SLAP - Simple Library for Automating Publication

Typical code workflow

  • Version Control - track changes
  • Feature Branches - isolate changes
  • Build Server - automatically test and deploy
  • Promotion - Integration => UAT => Production

We would probably never...

  • Make changes "live" on the server
  • Not keep our code in version control
  • Use a manual process for deployment

But this is our normal workflow for map services!

This is just a tooling problem!

  • ESRI has exposed almost everything via REST API
  • We just need some "high level" tools to use it
  • SLAP!
